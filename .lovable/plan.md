## 视觉增强方案

### 1. 新增动画效果 (tailwind.config.ts)

| 动画名称 | 效果描述 |
|---------|---------|
| `pulse-glow` | 柔和的金色光晕脉动效果 |
| `float-slow` | 缓慢的上下浮动 |
| `sparkle` | 闪烁的星光效果 |
| `slide-in-left` | 从左侧滑入 |
| `bounce-gentle` | 轻微的弹跳效果 |

### 2. 增强装饰组件 (CloudDecoration.tsx)

新增装饰元素:
- **灯笼装饰**: 左右两侧悬挂的灯笼SVG，带有轻微摇摆动画
- **烟花粒子**: 随机位置的小光点，带有闪烁效果
- **祥云增强**: 更多层次的云纹，不同大小和透明度
- **金色光斑**: 随机分布的金色光点，营造节日氛围

### 3. 首页优化 (Overview.tsx)

增强效果:
- 卡片添加 `hover:shadow-2xl` 和渐变边框光效
- 图标容器添加旋转微动画
- 标题文字添加金色渐变效果
- 新增 "恭贺新春" 装饰性横幅
- 卡片进入时添加错落有致的弹跳效果

### 4. 亲戚计算器页面 (Kinship.tsx)

增强效果:
- 输入框获得焦点时添加金色光晕
- 计算按钮添加脉动光效
- 结果展示添加展开动画
- 说明卡片添加悬停立体效果

### 5. 话术生成页面 (Dialog.tsx)

增强效果:
- 话题分类按钮添加图标装饰
- 选中状态添加粒子特效背景
- 话术卡片添加翻页式切换动画
- 复制成功添加撒花效果提示

### 6. 红包账本页面 (Ledger.tsx)

增强效果:
- 统计数字添加数字跳动动画效果
- 收入/支出用更鲜明的颜色区分（绿色上箭头/红色下箭头）
- 表格行添加交替背景色和悬停高亮
- 删除按钮添加确认动画
- 金额显示添加金币图标

### 7. 抽签页面 (Fortune.tsx)

增强效果:
- 签筒添加3D透视效果
- 摇晃动画增强，添加签条飞出效果
- 结果展示添加展开卷轴效果
- 不同签等级添加不同颜色光效（上上签金光闪烁）
- 添加祥云环绕的背景动画

### 8. 匿名信页面 (Email.tsx)

增强效果:
- 信纸背景添加复古纹理
- 输入框添加手写字体效果
- 发送按钮添加信封飞出动画
- 整体页面添加信纸泛黄效果

### 9. 全局样式增强 (index.css)

新增样式:
- `.glass-effect`: 毛玻璃效果
- `.gradient-text`: 金红渐变文字
- `.glow-shadow`: 发光阴影
- `.paper-texture`: 宣纸纹理背景
- `.lantern-glow`: 灯笼光晕效果

## 技术细节

```text
新增 Keyframes:
┌─────────────────┬────────────────────────────────────┐
│ pulse-glow      │ 0% → 50% → 100% 光晕明暗循环      │
│ float-slow      │ 0% → 50% → 100% Y轴位移循环       │
│ sparkle         │ 0% → 25% → 50% → 75% → 100% 闪烁  │
│ slide-in-left   │ -100% → 0% X轴位移               │
│ bounce-gentle   │ 带弹性的缩放效果                  │
│ unfold          │ 模拟卷轴展开                      │
└─────────────────┴────────────────────────────────────┘

新增 CSS 类:
┌─────────────────┬────────────────────────────────────┐
│ .glass-effect   │ backdrop-blur + 半透明背景         │
│ .gradient-text  │ 红金渐变 + background-clip: text   │
│ .glow-shadow    │ box-shadow 多层发光               │
│ .paper-texture  │ SVG 宣纸纹理背景                  │
│ .lantern-glow   │ radial-gradient 灯笼光效          │
└─────────────────┴────────────────────────────────────┘
```
